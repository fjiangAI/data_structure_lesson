# Lab 02 受限线性结构与串匹配

适用周次：Week 04-06

## 学习目标

- 用栈和队列解决典型约束问题。
- 理解循环队列 front/rear 的不变量。
- 实现 KMP 或朴素匹配并比较。

## 任务

1. 实现括号匹配，支持三类括号。
2. 实现循环队列并完成入队、出队、队空、队满判断。
3. 实现朴素串匹配和 KMP 匹配中的至少一种。
4. 用测试说明普通数组队列与循环队列的差别。

## 必测用例

- 空栈 pop 不应崩溃。
- 循环队列 rear 回绕后仍能保持逻辑顺序。
- 队满和队空不能混淆。
- 模式串为空或长于文本时要有明确行为。

## 提交要求

- C 源码文件和编译命令。
- 运行截图或终端输出。
- 对核心结构不变量的说明。
- 每个核心操作的时间复杂度。
- AI 使用记录：prompt 摘要、模型建议、人工核查、最终修改。

## 评分重点

结构设计 20%，核心操作 30%，边界处理 20%，复杂度说明 15%，测试证据 10%，代码风格 5%。
