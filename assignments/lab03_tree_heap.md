# Lab 03 树、AVL 与堆

适用周次：Week 07-09

## 学习目标

- 掌握二叉树递归遍历。
- 理解 AVL 旋转维护平衡的局部性。
- 用数组实现堆并完成上滤/下滤。

## 任务

1. 构造一棵二叉树并输出前序、中序、后序、层序遍历。
2. 实现 AVL 插入，至少支持 LL、RR、LR、RL 四种情况。
3. 实现最小堆 push、pop、peek 和 buildHeap。
4. 用小规模样例打印每次旋转或下滤后的状态。

## 必测用例

- 空树遍历应正常结束。
- AVL 插入 30,20,10 和 30,10,20 都应平衡。
- 堆中只有一个元素时 pop 正确。
- buildHeap 后每个父结点都满足堆序。

## 提交要求

- C 源码文件和编译命令。
- 运行截图或终端输出。
- 对核心结构不变量的说明。
- 每个核心操作的时间复杂度。
- AI 使用记录：prompt 摘要、模型建议、人工核查、最终修改。

## 评分重点

结构设计 20%，核心操作 30%，边界处理 20%，复杂度说明 15%，测试证据 10%，代码风格 5%。
