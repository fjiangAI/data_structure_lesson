# Lab 03 树、AVL 与堆 参考讲解

适用周次：Week 07-09

难度：中等

覆盖概念：遍历、AVL 旋转、堆、递归

## 教学意图

本 Lab 的核心不是得到 public output，而是让学生证明自己维护了正确的数据结构不变量。批改时应优先检查结构设计、边界处理和测试证据。

## Public 输出

```text
preorder: A B D E C F
avl-root: 20
heap-pop: 3 7 12
```

## 参考实现使用

- [reference_solution.c](reference_solution.c) 是同一公开头文件下的接口参考实现，可用于确认 public harness 和 hidden harness 工作正常。
- 它不替代课堂中的完整 ADT 讲解。教师讲解时应把每个函数拆回结构体字段、不变量和边界处理。

## Hidden tests 建议

- 遍历：补一个边界或反例测试，要求学生解释为什么能暴露错误。
- AVL 旋转：补一个边界或反例测试，要求学生解释为什么能暴露错误。
- 堆：补一个边界或反例测试，要求学生解释为什么能暴露错误。
- 递归：补一个边界或反例测试，要求学生解释为什么能暴露错误。

## 批改追问

1. 你的结构体字段各自维护什么关系？
2. 哪个操作最容易破坏不变量？
3. 最坏输入是什么，复杂度如何变化？
4. 如果 LLM 给出另一种实现，你如何用测试判断它是否可靠？
