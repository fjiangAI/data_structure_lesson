# 随堂测试题

每周题目可任选 3 到 5 题，要求学生在纸面或在线平台快速完成。建议至少包含 1 道手推题、1 道边界题和 1 道错误诊断题。


## Week 01 课程导论与算法分析基础

2. 二分查找在 32 个元素中最多比较几次？
3. 为什么 `O(1)` 不等于程序只执行 1 条语句？
4. 写出一个递归没有出口的风险。
5. 让模型解释复杂度时必须追问哪一个前提？
